# MixiRide

Ứng dụng cho thuê phương tiện thông minh với tích hợp bản đồ thời gian thực, định giá động và bảng điều khiển phân tích quản trị.

## Chạy ứng dụng

**Yêu cầu:** Node.js

1. Cài đặt dependencies:
   `npm install`
2. Thiết lập `VEHICLE_RENTAL_API_KEY` trong file [.env.local](.env.local) với API key của bạn
3. Chạy ứng dụng:
   `npm run dev`
